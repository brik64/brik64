# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.0.x   | Yes       |
| < 2.0.0 | No        |

## Reporting a Vulnerability

To report a security vulnerability, email **info@brik64.com** with subject `[SECURITY]`.

Include:
- Affected component and version
- Steps to reproduce
- Impact assessment

We will respond within 2 business days and aim for coordinated disclosure within 30 days.
